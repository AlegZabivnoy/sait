# Quiz Application

Система створення та проходження квізів з React та Webpack.

## Структура проекту

```
sait/
├── src/                    # Вихідний код для компіляції
│   ├── components/         # React компоненти
│   ├── css/               # CSS стилі
│   └── js/                # JavaScript модулі
├── public/                # Статичні HTML сторінки
│   ├── create/            # Створення квізу
│   ├── manage/            # Управління квізами
│   ├── quiz/              # Проходження квізу
│   └── results/           # Результати
├── dist/                  # Скомпільовані файли (генерується)
└── webpack.config.js      # Конфігурація Webpack

```

## Встановлення

1. Клонуйте репозиторій:
```bash
git clone https://github.com/AlegZabivnoy/sait.git
cd sait
```

2. Встановіть залежності:
```bash
npm install
```

## Запуск проекту

### Режим розробки (Development)

Запустити dev server на `http://localhost:3000`:

```bash
npm run dev
```

Або альтернативно:

```bash
npx webpack serve
```

Dev server автоматично перезавантажує сторінку при змінах в коді.

### Збірка для продакшну (Production)

Створити оптимізовану збірку в папці `dist/`:

```bash
npm run build
```

## Використання

1. **Головна сторінка** (`http://localhost:3000`) - перегляд доступних квізів
2. **Створення квізу** (`/create/`) - створення нового квізу з питаннями
3. **Управління квізами** (`/manage/`) - редагування та видалення квізів
4. **Проходження квізу** (`/quiz/`) - відповіді на питання квізу
5. **Результати** (`/results/`) - історія пройдених квізів

## Технології

- **React 19** - UI компоненти
- **Webpack 5** - збірка модулів
- **Babel** - транспіляція JSX
- **LocalStorage** - збереження даних

## Особливості

- Три типи питань: одна правильна відповідь, кілька правильних, розгорнута відповідь
- Збереження даних в localStorage (без бекенду)
- Responsive дизайн
- Hot Module Replacement в dev режимі

## Команди

```bash
npm run dev      # Запуск dev сервера
npm run build    # Збірка для продакшну
```
